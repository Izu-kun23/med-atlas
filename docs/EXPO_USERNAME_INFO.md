# Finding Your Expo Username

## Quick Method

Run this command in your terminal:

```bash
npx expo whoami
```

This will display your Expo username.

## Where It's Used

Your Expo username is used in:
- Google OAuth redirect URIs: `https://auth.expo.io/@izu/med-atlas`
- Expo project URLs
- Publishing to Expo

## If You're Not Logged In

If `npx expo whoami` shows you're not logged in, run:

```bash
npx expo login
```

Then enter your Expo account credentials.

## Alternative: Check Expo Dashboard

You can also find your username by:
1. Going to [expo.dev](https://expo.dev)
2. Logging in
3. Your username appears in the URL: `https://expo.dev/@your-username`
